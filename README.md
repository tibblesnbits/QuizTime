A simple interactive terminal-based multiple choice quiz I made while studying for cert exams. Built in Python.
Questions stored in a simple JSON file, empty format sample included, and a convert script. (I don't need compTIA coming for me sorry)

~ Multiple choice questions with shuffled answer order every run
~ Immediate right/wrong feedback with explanation after each question
~ Running score shown after every question
~ Final score with grade label at the end
~ Shuffle mode to randomize question order
~ Easy to add your own questions via JSON
